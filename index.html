HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Head element information mainly includes meta data such as character sets
  • HTML is best described as the frame around which the website is built, in opposition to CSS and JavaScript
  • The head element contains the title element, which appears at the tab we have open. it is incredibly useful for becoming more marketable to search engines
  • Body and head are examples of semantic elements, as opposed to div elements for example, which create a division
  • the a element stands for anchor, creating a link to another website. the href attribute points to the URL
  • The ul, li and ol elements represent lists
  • The p element represents a paragraph
  • H1 h2 etc represent the level of heading, with 1 being the largest and 6 the smallest
  • the br element creates a nice break
  • The img element contains information about the image displayed, the src attribute is the most important because it provides the location of the image shown

Git

  • li displays folders within your directory
  • CD "foldername" changes directory
  • CD.. changes directory to parent directory
  • git status checks what branch we are on
  • git checkout is essentially like checking out at a grocery store
  • The -b command indicates a new branch;ie git checkout -b "newbranchname"
  • git push uploads the update to github
  • git push --set-upstream indicates what you are pushing into
  • the origin command indicates location
  • git pull will download the most up to date version of our branch
  • git merge consolidates branches
  • the -m command adds a message, as in "git commit-m "message"
  • a commit must be staged, for which we use the git add "filename" command, if we want to undo the staging of a specific file, we would use git reset "filename"
  • git clone "url" will clone a repo from github down to our machine
  • git branch will list your branches, * will show which branch is active
  • git log will show all commits in the current branches history
